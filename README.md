## Ejecutar en modo desarrollo

1. Clonar el repositorio

```
git clone git@github.com:JhonEduard26/pokedex.git
```

2. Ejecutar

```
pnpm install
```

3. Tener Nest CLI instalado

```
npm install -g @nestjs/cli
```

4. Levantar la base de datos

```
docker-compose up -d
```

## Stack usado

- NestJS
- MongoDB
